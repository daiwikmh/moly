## Cast a Vote

```
You:  Vote YEA on proposal 177
```

**Simulation:**
```
Vote Simulation:
  Proposal: #177
  Vote:     YEA
  Note:     Voting requires LDO tokens at the snapshot block.
            Use MCP server in live mode to cast your vote.
```

**Live (MCP server):**
```
Vote Cast:
  Proposal: #177
  Vote:     YEA
  TX Hash:  0xdef...
```

## Requirements for Voting

1. **LDO tokens** — you must hold LDO at the proposal's snapshot block
2. **Open proposal** — can't vote on closed/executed proposals
3. **Live mode** — simulation mode only shows what would happen
